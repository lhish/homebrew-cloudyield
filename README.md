# homebrew-cloudyield

Homebrew tap for [CloudYield](https://github.com/lhish/CloudYield) - 让网易云音乐更智能

## 安装

```bash
brew tap lhish/cloudyield
brew install --cask cloudyield
```

系统要求：macOS 14.2+。首次运行需要授予「辅助功能」与「音频捕获」权限（用于控制网易云与检测其他应用出声）。

## 卸载

```bash
brew uninstall --cask cloudyield
brew untap lhish/cloudyield
```
